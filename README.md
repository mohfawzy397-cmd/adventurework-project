# 📊 AdventureWorks Sales Analysis Dashboard

## 📌 Overview
This project is a comprehensive sales analysis dashboard built using Power BI based on the AdventureWorks dataset.  
It provides insights into sales performance, customer trends, and product profitability.

---

## 🎯 Objectives
- Analyze company sales performance
- Evaluate product and category profitability
- Monitor customer buying patterns
- Track revenue growth over time

---

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel
- Power Query (Data Cleaning)
- DAX (Data Modeling & Calculations)

---

## 📈 Key Insights
- Total revenue and profit analysis
- Best and worst performing products
- Customer distribution and behavior
- Regional sales performance

---

## 📊 Dashboard Features
- Interactive slicers and filters
- KPI metrics (Revenue, Profit, Orders)
- Time-based trend analysis
- Product and category breakdown
